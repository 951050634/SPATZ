# Common LaTeX Assets

Shared material for multiple papers lives here.

```text
common/
├── bib/
│   └── refs.bib
└── figures/
    └── README.md
```

Rules:

- Put references shared by paper A and paper B in `common/bib/refs.bib`.
- Keep generated plots in their source data directories when possible.
- If a figure becomes paper-specific, document it in that paper's own README.
